The original bitmap file for the 8x8 pixel font I use was a screenshot from a FreeBASIC program I wrote.
FreeBASIC had its own graphics mode and included this font. I have never found a better font of this size that I liked more.

I named the file "FreeBASIC Font 8.bmp" as a reminder of its history.

When I was testing different image formats for my BBM project, this was a great test file to convert to other formats.
For example, I converted the file to a PBM (Portable Bitmap File).

convert "FreeBASIC Font 8.bmp" font8.pbm
